

⫷>[____GUI FEATURES____]<⫸

[__MENU__]
 Lets you use mods from the menu on the gui

[__LOBBY__]
> Info
 Shows info of the lobby you're in

> Joiner
 - (text box)
 - Join Room
 - Join Random Room
 - Join Random Room In Current Map
 - Join Random Priv Room
 - Rejoin Last Room
 - Change GameMode
 - Change Queue

> PlayerList
 Shows info about everyone. To the right of their info there are 2 buttons: "Cosmetics" & "More". "Cosmetics" shows all of their cosmetics, if equipped, category, and when they bought them. "More" shows extra info of the player. In More if you click Props it acts as a basic mod checker.

> Name & Color
 - (text box)
 - Set Name
 - (red slider)
 - (green slider)
 - (blue slider)
 - Set Color
 - Change Turn Type
 - Change Turn Speed
 - Change Mic Input

> My Cosmetics
 Shows all the cosmetics you own. Clicking one will toggle the cosmetic

[__OTHER__]
> WASD
 - WASD Enabled
 - WASD No Gravity
 - WASD Speed Changer
 (Controller Emulator)
 - L/R Trigger
 - L/R Grip
 - A/B/X/Y
 - L/R Joystick
 - Emulate PC Gun
 - First Person Camera

> GUI Settings
 - Button Color
 - Text Color
 - Border Color
 - Background Color
 - Solid Background
 - Draggable GUI
 - Save Settings
 - Load Settings

> Keybinds
 [ (Left Bracket) = Recenter GUI
 \ (Backslash) = Disable All Mods
 Insert (FN + Delete) = Toggle GUI
